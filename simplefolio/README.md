From your command line, first clone Simplefolio:

Using NPM: Simply run the below commands.

bash
# 2022 Update - Fix Dependencies
$ npm audit fix
$ npm i @parcel/transformer-sass

# Install dependencies
$ npm install

# Start the development server
$ npm start
